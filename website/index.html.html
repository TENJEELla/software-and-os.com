COMPUTER SOFTWARE AND OPERATING SYSTEM

Concept of software

Defination

A set of programs written for a computer to performe a particular task is called software. The logical components or set of procedures to routines or instructions are called software.Software is a set of instructions, data or programs used to operate computers and execute specific tasks. It is the opposite of hardware, which describes the physical aspects of a computer. Software is a generic term used to refer to applications, scripts and programs that run on a device. It can be thought of as the variable part of a computer, while hardware is the invariable part.Early software was written for specific computers and sold with the hardware it ran on. In the 1980s, software began to be sold on floppy disks, and later on CDs and DVDs.Software can be found on vendor websites or application service provider websites.

Categoires of software

The following are the categoires of software:
  1. System software
  2. Application software
  3. Utility software
  4. Web base software
  5. Mobile apps(applicatons)
